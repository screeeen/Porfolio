Interaction Design

I apply user centered techniques keeping it simple. I got experience on videogames player psychology, user interfaces creation I use storytelling as a vehicle to explain abstract concepts to users.

Game Design

Independent Games

My indie games development is my sandbox to learn coding and try new ideas. I started doing games with game maker, then moved to Unity and learn some c#. I released two games on iOS and Android. I learnt a lot about how to market a game on my own. Now I do little browser games for jams to grasp javascript skills and serverside experiments. It's an ongoing process that I like to enjoy with tea or coffee.

Virtual Reality

I learnt how to set up an intuitive, non motion sickness, reliable VR experience by attending jams. My last project in this field consisted of an entertainment platform for airline passengers.

The framework is a compound of a VR client, a stakeholders tool for 3rd party media and a cabin crew device for comunications with passengers. The conception was made in a low fidelity setting and the minimum viable product was tested sucessfully in several different airlines such Iberia or Lufthansa amongst others. Currently the project is comercially deployed, up and running.
